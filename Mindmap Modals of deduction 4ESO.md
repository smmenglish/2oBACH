# Modals of deduction

## About
### Funcionan como cualquier modal <mark style="background: #FFF3A3A6;">(<span style="color:rgb(0, 112, 192)">**Subject**</span> + <span style="color:rgb(255, 0, 0)">**modal verb**</span> + <span style="color:rgb(175, 44, 127)">**verb (infinitive)**</span> )</mark>
- <span style="color:rgb(0, 112, 192)">Mark</span> <span style="color:rgb(255, 0, 0)">must</span> <span style="color:rgb(175, 44, 127)">be</span> in Madrid.
- <span style="color:rgb(0, 112, 192)">Alina</span> <span style="color:rgb(255, 0, 0)">can't</span> <span style="color:rgb(175, 44, 127)">be</span> serious.
### El uso depende de lo que queramos expresar.
### Normalmente van <mark style="background: #FFB86CA6;">acompañados de una frase</mark> que nos ayuda a entender qué modal usar.
- <mark style="background: #FFB86CA6;">It is snowing in Reinosa</mark>. <span style="color:rgb(0, 112, 192)">It</span> <span style="color:rgb(255, 0, 0)">must</span> <span style="color:rgb(175, 44, 127)">be</span> really cold.
-  <span style="color:rgb(0, 112, 192)">Mark</span> <span style="color:rgb(255, 0, 0)">might not</span> <span style="color:rgb(175, 44, 127)">finish</span> the task on time. <mark style="background: #FFB86CA6;">He is working very slow</mark>.



## Must
### Lo usamos cuando estamos seguros de algo en afirmativa.
- The lights are on. <span style="color:rgb(0, 112, 192)">Anna</span> <span style="color:rgb(255, 0, 0)">must</span> <span style="color:rgb(175, 44, 127)">be</span> at home.
- She gets excellent grades. <span style="color:rgb(0, 112, 192)">She</span> <span style="color:rgb(255, 0, 0)">must</span> <span style="color:rgb(175, 44, 127)">be</span> a very good student.

## Can't
### Lo usamos cuando estamos seguros de algo en negativa.
- The office is empty and the lights are off. <span style="color:rgb(0, 112, 192)"><span style="color:rgb(0, 112, 192)">Mr. Brown</span></span> <span style="color:rgb(255, 0, 0)">can't</span> <span style="color:rgb(175, 44, 127)">be</span> inside.
- Tom? He never watches football matches. <span style="color:rgb(0, 112, 192)"><span style="color:rgb(0, 112, 192)">He</span></span> <span style="color:rgb(255, 0, 0)">can't</span> <span style="color:rgb(175, 44, 127)">be</span> a fan.

## Might / Might not
### Lo usamos para expresar duda en afirmativa o negativa.
- A menudo se usa junto a expresiones como "**maybe**", "**perhaps**", "**it's possible**", "**not sure**"
- Dark clouds are coming from the sea. <span style="color:rgb(0, 112, 192)">It</span> <span style="color:rgb(255, 0, 0)">might</span> <span style="color:rgb(175, 44, 127)">rain</span> in the afternoon.
